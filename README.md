# Lernspiele für wissenschaftliches Arbeiten

Eine Sammlung SCORM basierter Lernspiele rund um wissenschaftliches
Arbeiten, entwickelt mit KI Unterstützung von Studierenden des
berufsbegleitenden Masterstudiengangs E-Learning und Wissensmanagement
(MEWM) an der Hochschule Burgenland.

## Was sind diese Spiele

Die Spiele sind im Rahmen einer Lehrveranstaltung von Barbara Geyer im
Masterstudiengang MEWM entstanden. Die Studierenden haben sie mit
Unterstützung von KI Tools konzipiert und entwickelt. Jedes Spiel liegt
als fertiges SCORM Paket vor und lässt sich direkt in Moodle oder andere
SCORM fähige Lernmanagementsysteme einbinden. Keine Installation, kein
Login. Alle Spiele sind frei verwendbar unter offener Lizenz, MIT für
den Code und CC BY 4.0 für die Inhalte.

Begleitende Konzeptdokumente, Anleitungen und didaktische Reflexionen
finden sich auf dem Padlet zum Projekt:

 [Padlet "Lernspiele für wissenschaftliches Arbeiten"](https://padlet.com/innovative_lehre/lernspiele-fur-wissenschaftliches-arbeiten-oi8zni662ow0ww74)

## Spiele im Unterricht nutzen

Jedes Spiel liegt in diesem Repository als SCORM Paket (ZIP) bereit und
kann in der eigenen Lehre auf zwei Arten genutzt werden.

**In Moodle als SCORM Aktivität einbinden.** Auf der GitHub Seite des
jeweiligen Spiels die SCORM ZIP Datei herunterladen. In Moodle den Kurs
öffnen, Bearbeiten einschalten, **Aktivität hinzufügen → Lernpaket
(SCORM)**, die ZIP Datei hochladen und speichern. Vorteil gegenüber
einem einfachen Link: Lernfortschritt und Punktestände werden
automatisch erfasst, das Spiel lässt sich bewerten und in Lernpfade
integrieren.

**Anpassen und weiterverwenden.** Das SCORM Paket entpacken, HTML, CSS
und JavaScript verändern, unter Namensnennung weiterverwenden. Alles
erlaubt.
